## üöÄ Getting Started

### Prerequisites

- Java 21 JDK
- Docker
- Docker Compose

### Local Development Setup

1. Clone the repository
   ```bash
   git clone https://github.com/[YOUR_USERNAME]/eventlinkr-backend.git
   cd eventlinkr-backend
   ```

2. Build the project
   ```bash
   ./mvnw clean install
   ```

3. Run with Docker Compose
   ```bash
   docker-compose up --build
   ```
